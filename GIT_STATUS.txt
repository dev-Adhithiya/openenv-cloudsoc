AUTHENTICATION HELP
================================================================================

If you need to set up GitHub authentication:

Option 1: Personal Access Token (Recommended)
  1. GitHub → Settings → Developer settings → Personal access tokens
  2. Generate new token with "repo" scope
  3. Use token as password when pushing

Option 2: SSH Keys
  1. Generate: ssh-keygen -t ed25519 -C "your@email.com"
  2. Add to GitHub: Settings → SSH Keys
  3. Use: git@github.com:USERNAME/openenv-cloudsoc.git

Option 3: GitHub CLI
  1. Install: https://cli.github.com
  2. Run: gh auth login
  3. Follow prompts
